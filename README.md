# ui-component-collection
**커스텀 UI 컴포넌트 컬렉션 플레이그라운드**

## Tech Stack
Node v14 base
- Vue v2
- Webpack v5
- Tailwind v2

## Usage
**Install dependencies**
```
npm install
```

**Run development mode (only)**
```
npm run dev
```
